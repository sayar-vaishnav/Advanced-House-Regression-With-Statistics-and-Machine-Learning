# Week 2 – Data Quality Assessment, Missing Value Analysis & Statistical Validation

## Project Objective

This phase focused on evaluating the quality of the Ames Housing dataset before feature engineering and model development. The objective was to understand missing data patterns, apply domain-driven imputation strategies, and validate important relationships using statistical inference.

---

## Missing Value Analysis

A detailed feature-wise investigation was conducted to distinguish between **true missing values** and **missing values representing the absence of house amenities**. Features such as **PoolQC, Alley, Fence, FireplaceQu, Garage*, and Basement*** were identified as meaningful "Not Available" cases rather than incomplete observations.

Numerical features, including **LotFrontage, MasVnrArea, GarageYrBlt, GarageArea, and GarageCars**, were analyzed individually before selecting suitable imputation methods.

---

## Missing Value Handling

Imputation strategies were chosen based on both statistical evidence and domain knowledge rather than applying a single method across all features.

* Missing categorical values representing absent amenities were replaced with appropriate **"No Feature"** categories.
* Low-percentage categorical missing values (e.g., **Electrical**) were imputed using the mode.
* Numerical features were evaluated individually before applying suitable imputation techniques.

This approach preserved business meaning while minimizing unnecessary bias.

---

## Statistical Validation

Statistical tests were performed to validate relationships before feature engineering.

### Welch's Two-Sample t-Test

House prices were compared between high-quality and low-quality homes.

**Results**

* **t-statistic:** 28.21
* **p-value:** < 0.001

The analysis confirmed that **OverallQual** has a highly significant impact on **SalePrice**, making it one of the strongest predictors in the dataset.

### Confidence Interval Analysis

95% confidence intervals showed a clear separation between both quality groups, reinforcing the practical significance of the observed difference.

### Association Analysis

**Cramér's V** was used to measure the strength of association between categorical variables, providing additional evidence before preprocessing decisions.

---

## Key Findings

* Missing values were carefully interpreted instead of being treated uniformly.
* Domain knowledge guided feature-specific imputation strategies.
* Construction quality (**OverallQual**) showed a strong statistically significant relationship with house prices.
* Confidence intervals supported the magnitude of this relationship.
* Statistical analysis was used to justify preprocessing decisions before feature engineering.
